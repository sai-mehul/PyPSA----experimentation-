Project 1 : Single Snapshot Economic Dispatch
Simulates a single-bus network where a Wind Turbine (cheap) and Gas Turbine (expensive) meet a 100 MW load in one hour. Demonstrates merit order dispatch cheapest generator runs first.
Project 2 : 24:Hour Wind Profile Simulation
Extends Project 1 to 24 hourly snapshots with a timen varying wind capacity factor. Shows how dispatch changes across the day as wind availability fluctuates.
Project 3: Battery Storage Integration
Adds a 50 MW / 200 MWh battery to the network. Battery charges during high wind hours and discharges when wind drops, reducing reliance on the expensive Gas Turbine.
