• Project 1: Single Snapshot Economic Dispatch

Simulated a single-bus power network with a Wind Turbine (low cost) and Gas Turbine (high cost) supplying a 100 MW load for a single hour.
Demonstrated merit order dispatch, where the cheapest generator is dispatched first.

• Project 2: 24 Hour Wind Profile Simulation

Extended the single snapshot model to 24 hourly snapshots using a time-varying wind capacity factor.
Analyzed how generator dispatch changes throughout the day as wind availability fluctuates.

• Project 3: Battery Storage Integration

Added a 50 MW / 200 MWh battery storage system to the network.
Simulated battery charging during high-wind periods and discharging during low-wind periods to reduce dependence on the expensive Gas Turbine.

• Project 4: Monte Carlo Simulation

Added a simulation concept with the VaR aspect to show the costs incurred vary via uncertanity of wind conditions
